# Stress-Analysis-of-a-Block
The stress analysis of a 3-Dimensional block under a normal load was analysed
The Shear stresses, octahedral stress and the von mises stresses were calculated for the block at various points inside the block after the load was applied using the displacement of the points.
A Numpy code was created to help in the analysis of the block.
